- health-check.sh

## Purpose
These scripts were created to support day-to-day operations for the EC2 + Nginx project.

## Main operational areas covered
- Site health verification
- Backup creation
- Log inspection
- Safe Nginx reloads
- Simple custom log rotation

## Notes
The scripts are designed for learning and portfolio use, and show basic operational discipline for a Linux web server project.
