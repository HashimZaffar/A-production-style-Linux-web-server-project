# Deployment Checklist

## Before launch
- AWS account accessible
- Billing method active
- Region selected
- Project folder created
- Git repository initialized
- .gitignore created
- README created
- AWS planning documents created

## At launch
- Create key pair
- Choose Ubuntu image
- Choose small instance type
- Use default VPC
- Assign public IP
- Create dedicated security group
- Allow SSH, HTTP, and HTTPS
- Launch instance

## After launch
- Connect by SSH
- Update packages
- Create admin user
- Harden SSH configuration
- Install Nginx
- Test public access
- Add domain
- Enable HTTPS
- Add Bash scripts
