# AWS Plan

## Cloud platform
AWS

## Core service
Amazon EC2

## Project goal
Build and operate a production-style Linux web server for learning and portfolio use.

## Planned operating system
Ubuntu LTS

## Planned instance type
t3.micro

## Planned network approach
Use the default VPC and public subnet for the first version of the project.

## Planned access method
SSH using an AWS key pair

## Planned web server
Nginx

## Planned public ports
- 22 for SSH
- 80 for HTTP
- 443 for HTTPS

## Planned future tasks
- Launch EC2 instance
- Connect with SSH
- Harden Linux access
- Install and configure Nginx
- Attach domain or subdomain
- Enable HTTPS with Let's Encrypt
- Add Bash scripts for backup, health checks, log rotation, and restarts
