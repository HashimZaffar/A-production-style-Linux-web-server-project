# first-project

A beginner-friendly DevOps portfolio project to build and operate a production-style Linux web server on AWS.

## Project goal

This project demonstrates practical skills in:

- Linux server administration
- AWS EC2 setup
- SSH hardening
- Nginx configuration
- DNS and domain mapping
- HTTPS with Let's Encrypt
- Bash scripting for operations
- Basic troubleshooting and service management

## Planned project phases

1. Project foundation and local Git setup
2. AWS account readiness and project planning
3. Launching the EC2 instance
4. SSH access and Linux hardening
5. Installing and configuring Nginx
6. Domain/subdomain connection
7. HTTPS with Let's Encrypt
8. Bash scripts for server operations
9. Final cleanup, testing, and documentation
